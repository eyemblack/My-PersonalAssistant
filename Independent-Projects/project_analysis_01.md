# Project: Corporate PR Website
## Client Requirements Analysis

- **CMS (ระบบหลังบ้าน):** Required (High effort) -> แนะนำให้ใช้ CMS สำเร็จรูปหรือ Framework ที่จัดการง่าย
- **PR News:** 3-4 times/month + Search functionality (Medium effort)
- **Assets:** Client provided (Low effort)
- **Contact Form:** Email notification (Low effort)
- **Hosting/Domain:** Client is confused (Requires explanation)

## Recommended Stack (Efficiency Focus)
- **Frontend:** Next.js (Fast, SEO friendly)
- **CMS:** Contentful หรือ Sanity (Headless CMS) - ลูกค้าใช้ง่าย และฟรีใน Tier เริ่มต้น
- **Hosting:** Vercel (Free/Low cost)
- **Email:** Resend หรือ SendGrid
